RSA description

Type: Asymmetric cipher.

Strength: Strong when large keys are used; small keys (like this demo) are easy to break.

How it works: Pick primes p and q; n = p×q and φ(n) = (p−1)(q−1). Choose public exponent e; private exponent d satisfies e×d ≡ 1 (mod φ(n)). Encrypt: C ≡ M^e (mod n). Decrypt: M ≡ C^d (mod n). Security rests on the difficulty of factoring n.

Key size: 2048–4096 bits are typical in production (TLS, signatures, key exchange). This page uses tiny primes for speed.
