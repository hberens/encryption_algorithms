Computer security — cipher lab

This project implements several ciphers in a web interface. You can type a message or upload a file (for 3DES, AES, and RSA), import keys from a file, and choose whether to show the result on the page or download it as a file. Vigenère accepts text only for the message (as required for that cipher).

  • Vigenère — polyalphabetic text cipher.
  • 3DES — triple DES on binary data.
  • AES — AES-CBC (pure Python core).
  • RSA — textbook RSA demo (small keys).
